# MHM Atelier

A portfolio-ready project built for GitHub Pages, resume showcase, and creative presentation. `MHM Atelier` is designed as a polished portfolio site with a responsive layout, interactive work cards, and an AI assistant for instant recruiter guidance.

## What’s included

- `index.html` — the main portfolio page with sections for Home, About, Projects, Skills, AI Bot, and Contact.
- `style.css` — a modern dark glassmorphism design system with gradients, responsive grid layouts, and accessible typography.
- `script.js` — interactive navigation, responsive menu behavior, and live package pricing updates for each project card.
- `chatbot.js` — a lightweight AI assistant widget that responds to portfolio, hiring, and design-related queries.

## Features

- attractive hero section with brand identity
- sticky navigation and smooth scrolling
- featured work section with selectable package tiers
- skills grid and contact section for hiring-ready presentation
- AI bot for recruiter and collaborator support
- optimized for GitHub Pages deployment

## How to use

1. Open `index.html` in your browser.
2. Click navigation items to jump through the portfolio sections.
3. Use the project package selectors to see live pricing updates.
4. Ask the AI assistant questions in the chat widget for on-page guidance.

## Customization

- Replace the Lorem-style text content with your own experience, project descriptions, and contact details.
- Swap the image URLs in `index.html` with your own project visuals.
- Update the email and LinkedIn link in the Contact section.
- Adjust the package base price and pricing tiers in `script.js`.

## Deployment

For GitHub Pages:

1. Push the repository to GitHub.
2. In repository settings, enable GitHub Pages from the `main` branch.
3. Set the site folder to `/`.
4. Your portfolio will be live on the GitHub Pages URL.

## Notes

This project is a fully connected portfolio package suitable for a resume presentation, including a dynamic AI assistant that enhances user engagement. Feel free to personalize it as your own creative brand identity.
